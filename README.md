# Instagram 養號工具

基於 Android AccessibilityService 的智能 Instagram 養號 APK。

## 功能

- 首頁信息流自動滾動瀏覽
- 關鍵詞搜索瀏覽（支持愛馬仕包包等關鍵詞）
- Reels 自動觀看
- 混合模式智能切換
- 隨機點讚（可調概率）
- 廣告檢測跳過
- 防誤觸保護

## 使用方式

1. 從 GitHub Actions 下載編譯好的 APK
2. 安裝到雲手機/矩陣機
3. 打開 App，授予無障礙服務權限
4. 配置模式與關鍵詞，點擊「開始養號」

## 自動構建

每次推送代碼到 `main` 分支，GitHub Actions 會自動編譯 Debug APK。

前往 Actions 頁面查看構建狀態並下載 APK。
